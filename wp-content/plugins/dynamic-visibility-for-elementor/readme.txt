Extension for Elementor, you can hide elements by some rule (role, date, ecc)

== Description ==

This widget lets you decide when to hide or show any widgets in your pages.
It’s an Elementor add-on, so you must install Elementor FREE before activating the plugin.

- Set a date (from – to) and decide when each widget will be visible.
- Limit visibility for specific user roles or users or simply hide elements for future publishing.
- Set a fallback text (e.g.: ‘Coming soon’) for hidden elements that will be displayed in place of the widgets.

[View FULL FEATURES, DEMO and MORE](https://www.dynamic.ooo#dynamic-visibility)

We also suggest this amazing plugin: [Dynamic Animations for Elementor](https://wordpress.org/plugins/dynamic-animations-for-elementor/)

= How it works =

Simply open a page in Elementor mode. Select your widget, go to Advanced tab.
You'll find a new “Dynamic” configuration. Open it, enable Visibility and set as you prefer.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/dynamic-visibility-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There is nothing to configure. It's necessary have installed and activated Elementor FREE
